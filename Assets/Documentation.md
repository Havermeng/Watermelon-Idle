# Документация проекта Watermelon Idle

## Общая информация
- **Название проекта**: Watermelon Idle
- **Язык**: C# (Unity)
- **Платформа**: WebGL, мобильные устройства

## Ключевые системы

### SaveManager
- **Назначение**: Управление системами сохранения игры, включая слоты, данные игрока и операции сохранения/загрузки.
- **Основные методы**:
  - `SaveToSlot(int slot)` – сохраняет текущий прогресс в указанный слот.
  - `LoadFromSlot(int slot)` – загружает прогресс из слота.
  - `DeleteSlot(int slot)` – удаляет сохранение слота.
  - `CreateNewSave(int slot, string name)` – создает новое пустое сохранение.
  - `RenameSlot(int slot, string newName)` – переименовывает слот.
- **Файлы**: `Assets/Scripts/Menu/SaveManager.cs`

### MenuManager
- **Назначение**: Управление меню пользовательского интерфейса, включая анимацию переходов между панелями.
- **Основные методы**:
  - `OpenSaveSlots()` – открывает панель слотов сохранения.
  - `OpenSettings()` – открывает панель настроек.
  - `BackToMainMenu()` – возвращает в главное меню после анимации.
  - `QuitGame()` – выход из игры.
- **Анимации**: Слайд-анимации с использованием `SlideTo`, `SlideBack`.
- **Файлы**: `Assets/Scripts/Menu/MenuManager.cs`

### WebGLTitleHider
- **Назначение**: Скрытие объектов в WebGL-сборке игры при необходимости.
- **Параметры**:
  - `hideInWebGL` – флаг, определяющий, следует ли скрывать объект в WebGL.
- **Логика**: При запуске проверяет `#if UNITY_WEBGL && !UNITY_EDITOR` и скрывает объект, если `hideInWebGL` установлен в `true`.
- **Файлы**: `Assets/Scripts/Menu/WebGLTitleHider.cs`
